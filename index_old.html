Below are the common commands that check:
    \du              # List all users and roles, and privileges
    \l               # To list all the DBs in the SQL-Server
    \dt              # List all tables
    \d               # List all relations
    SELECT current_database();  # Check your current DB

# To create insert 
    \i /path/to/your/file.sql

There are 2 levels in DB login:
    1. Level 1: The DB_server, which lets you see all the DBs in the DB-Server.
    2. Level 2: Inside a specific DB, e.g., northwind.

    \q      # Exit from level 2
    exit    # Exit from level 1

# To run SQL:
    SELECT * FROM customers;

# To exit out of the query data:
    q
